🧠 AI-Blog Pro: Next-Gen AI-Powered Blogging Platform
AI-Blog Pro is a high-performance MERN stack application integrated with Google Gemini 1.5 Flash AI. It is designed to empower technical writers by providing AI-driven content suggestions, automated meta-data generation, and secure cloud storage.

✨ Key Features
✨ AI Writing Assistant: Uses the Gemini 1.5 Flash API to generate SEO-friendly titles and extend technical paragraphs based on initial prompts.

☁️ Cloud Persistence: Fully integrated with MongoDB Atlas for secure and permanent storage of blog documents.

📊 SEO Analytics: Real-time feedback on content readability and optimization.

💻 Modern UI: Responsive Glassmorphism design built using Tailwind CSS.

🛠️ Technical Stack
Frontend: HTML5, Tailwind CSS, JavaScript (ES6+).

Backend: Node.js, Express.js.

Database: MongoDB Atlas (Cloud NoSQL).

AI Integration: Google Gemini 1.5 Flash API.

DevOps: Render (Backend), GitHub Pages (Frontend).
